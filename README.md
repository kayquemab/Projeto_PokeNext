# 📌 PokeNext

Aplicação web com Next.js que consome a PokéAPI para exibir dados de Pokémon.

## 🚀 Tecnologias

* Next.js
* Tailwind CSS
* JavaScript

## ✨ Funcionalidades

* Listagem de Pokémon
* Página de detalhes
* Busca e filtros
* Navegação com rotas dinâmicas

## ⚙️ Como executar

```bash
git clone https://github.com/kayquemab/Projeto_PokeNext
cd PokeNext

# Instalar dependências
npm install

# Executar o projeto
npm run dev
```

## 📌 Observações

* Requer Node.js instalado
* Consome dados da PokéAPI
